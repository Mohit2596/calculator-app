# calculator-app
A basic calculator web app built using HTML, CSS, and JavaScript that performs simple arithmetic operations.



##  Features

* Addition
* Subtraction
* Multiplication
* Division
* Clear display (C button)
* Delete last input
* Fast and responsive design
* Error handling for invalid expressions



## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
